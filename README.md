# 📰 briefr - React Native App

A responsive mobile news application built with React Native and Expo. It allows users to browse the latest headlines across multiple categories with a clean and structured UI.

[![React Native](https://img.shields.io/badge/React_Native-20232A?style=for-the-badge&logo=react&logoColor=61DAFB)](https://reactnative.dev/)
[![Expo](https://img.shields.io/badge/Expo-000020?style=for-the-badge&logo=expo&logoColor=white)](https://expo.dev/)
[![NewsAPI](https://img.shields.io/badge/NewsAPI-FF4B4B?style=for-the-badge&logo=rss&logoColor=white)](https://newsapi.org/)

---

## ✨ Features

- Category-based news browsing (General, Business, Entertainment, Health, Science, Sports, Technology)
- Card-based UI for article display
- Automatic image thumbnails for articles
- Pull-to-refresh for latest headlines
- Error and empty state handling
- Open full articles via external links

---

## 🛠️ Tech Stack

- React Native (Expo)
- `@expo/vector-icons` (Feather, MaterialIcons)
- NewsAPI

---

## 🚀 Getting Started

### Prerequisites

- Node.js installed
- Expo account (optional, for Expo Go)
- API key from [newsapi.org](https://newsapi.org/)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/RNnews-app.git
cd RNnews-app
```

2. Install dependencies:
```bash
npm install
```

3. Start the app:
```bash
npm start
```
